How to get data into Google Sheets
1.Download Orders CSV and Expenses CSV below.
2.Open sheets.new to create a new Google Sheet.
3.Rename "Sheet1" → Drink Orders. Go to File → Import and upload the orders CSV.
4.Click + to add a new tab, rename it Expenses, and import the expenses CSV.
5.You're done! Both tabs are live in Google Sheets. ☕
